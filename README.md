Now, for Shortcuts
```
Virtual Desktops -> Cmd + 1,2,3,4 till 9 , ex Cmd+1
To move a single(window) Node to another desktop (say V.desktop 2 ) , ex cmd + shift + 2
To move a single(window) Node to another desktop AND user to move to that desktop as well (say V.desktop 2 ) , ex cmd + ctrl + 2

To move Focus of a window in a single tile ->  cmd + h(left) or j(down) or k(up) or l(right)
To move window completely -> cmd + shift + h or j or k or l 
To Organize everything quickly -> ctrl + o

For more refer the toml file 

```
